Configuración inicial — Firebase
Paso 1 — Crear proyecto Firebase
1. Andá a console.firebase.google.com
2. Clic en "Crear un proyecto"
3. Nombre: camara-ctrl → Continuar
4. Desactivá Google Analytics → Crear proyecto
5. En el menú izquierdo: Realtime Database → Crear base de datos
6. Elegí región us-central1 → Siguiente
7. Seleccioná "Empezar en modo de prueba" → Listo
8. En el menú izquierdo: Configuración del proyecto (⚙) → General
9. Bajá hasta "Tus apps" → clic en el ícono </> (Web)
10. Nombre de la app: camara → Registrar app
11. Copiá los datos del bloque firebaseConfig
Paso 2 — Pegá tu configuración Firebase
Paso 3 — Configurá usuarios y PIN
El PIN tiene 4 dígitos. Cada usuario tiene su nombre y el mismo PIN compartido (podés cambiarlo después).
Usuarios del sistema:
Paso 4 — Reglas de seguridad Firebase
En Firebase Console → Realtime Database → Reglas, pegá esto:
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
Nota: el PIN en la app protege el acceso visual. Las reglas de Firebase controlan quién puede leer/escribir a nivel de base de datos.
